# Speed Reader

A modern, premium RSVP (Rapid Serial Visual Presentation) speed reading application built with React + Vite.

![Speed Reader Screenshot](https://img.shields.io/badge/React-18-blue) ![Vite](https://img.shields.io/badge/Vite-5-purple) ![License](https://img.shields.io/badge/License-MIT-green)

## âœ¨ Features

- **RSVP Reading** - Display words one at a time for faster reading
- **Adjustable Speed** - 60-2000 WPM with presets (300, 500, 900, 1800)
- **Focus Highlighting** - Optimal Recognition Point (ORP) letter highlighting in red
- **Progress Tracking** - Visual progress bar with word count
- **Font Customization** - Adjustable size and font family (Sans, Serif, Mono)
- **Theme Switcher** - Dark, Light, and System themes
- **File Upload** - Drag & drop .txt files
- **Keyboard Shortcuts** - Full keyboard control
